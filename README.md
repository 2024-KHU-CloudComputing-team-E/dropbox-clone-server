# Dropbox Clone Project 📥 - INSTABOX
Dropbox의 파일 업로드 다운로드 등의 기본적인 기능을 포함한 Instagram의 SNS 성격이 더해진 웹 서비스입니다.

소셜 로그인을 지원하며, 사진을 AI 태그로 분류하는 기능과 다운로드 시 즐길 수 있는 간단한 게임을 제공합니다.
